# Honkpad v1.0.12 Release

**A soundboard for streamers, gamers, and anyone who wants professional audio routing.**

## What's New in v1.0.12

- 🔐 **Self-signed certificate** — App now shows "Jader Moura" as publisher instead of "Unknown Publisher"
- 🌍 **Full translations** — English, Portuguese (Brazil), Spanish
- 🎯 **Complete audio routing** — Separate control for stream output, monitor speaker, and microphone
- ⌨️ **Custom hotkeys** — Keyboard shortcuts for instant sound playback
- 🎵 **Sound groups** — Organize sounds into categories

## Downloads

- **Installer** (Recommended): `Honkpad Setup 1.0.12.exe` — Full installation with Start Menu shortcuts
- **Portable**: `Honkpad 1.0.12.exe` — Run directly, no installation needed

## ⚠️ Windows SmartScreen Warning

**If you see a Windows warning that says "Windows protected your PC":**

This is normal for new applications. Your app now shows "Jader Moura" as the trusted publisher instead of "Unknown Publisher".

**To bypass the warning:**

1. **Click "More info"** on the SmartScreen dialog
2. **Click "Run anyway"** 
3. The installer will launch normally

After 2-3 months of downloads, Windows will trust it automatically.

## Features

✅ Import audio files (MP3, WAV, OGG, M4A, FLAC, AAC)  
✅ Search MyInstants for pre-made sounds  
✅ Create sound groups for organization  
✅ Dual audio output (stream + monitor)  
✅ Microphone passthrough to virtual output  
✅ Custom hotkeys for sounds and groups  
✅ Multi-language support  
✅ Search and organize sounds instantly  

## Installation Issues?

**Windows blocks the installer?** See the SmartScreen section above.

**Still having trouble?** Report it in Issues with:
- Windows version (run `winver` to check)
- What device you're using as "Monitor (You)" in Settings
- Screenshot of any error messages

## Getting Started

1. Download and install
2. Go to Settings → Configure your audio devices
3. Import some sounds
4. Assign hotkeys
5. Start using!
